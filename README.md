# 🎓 EduPro Analytics Dashboard

## Project Overview
Data-driven Instructor & Course Quality Evaluation Platform

## Features
- Interactive Dark Theme Dashboard
- 6 Analytical Charts
- Sidebar Filters
- KPI Cards
- Instructor Leaderboard

## Tech Stack
- Python
- Pandas
- Plotly
- Streamlit

## Key Findings
- 2 Teachers = 60% of all enrollments
- Experience-Rating Correlation = 0.598
- Marketing = Best rated category
- Machine Learning = Lowest rated category

## How to Run
pip install streamlit pandas plotly
streamlit run dashboard.py
